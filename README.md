# 🌾 Aata Mitra - Grocery App

**Aata Mitra** ek simple aur fast grocery ordering app hai jo specifically flour (aata) aur daily essentials ke liye banayi gayi hai. Ye app Google Sheets ko as a database use karti hai.

## 🚀 Features
- **Live Inventory:** Google Sheets se real-time products load hote hain.
- **Easy Ordering:** Customer apna naam aur number daal kar seedha order place kar sakta hai.
- **Google Sheets Integration:** Saare orders aur stock automatically manage hote hain.
- **Mobile Friendly:** Ise phone par 'Add to Home Screen' karke app ki tarah use kiya ja sakta hai.

## 🛠️ Tech Stack
- **Frontend:** HTML5, CSS3, JavaScript
- **Backend:** Google Apps Script (GAS)
- **Database:** Google Sheets

## 📋 Setup Instructions
1. Google Sheet mein `Products`, `Orders`, aur `Customers` tabs banayein.
2. Google Apps Script ko deploy karke Web App URL generate karein.
3. `index.html` mein `scriptURL` variable ko apne URL se update karein.
4. GitHub Pages par host karein.
